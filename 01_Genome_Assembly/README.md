## 二代基因组组装：基于 k-mer 评估的策略优化

- **数据**：拟南芥 paired-end PE150（约 1.1 Gb）
- **方法**：Jellyfish + GenomeScope 预估基因组大小、杂合度、重复序列比例
- **优化**：对比 k=19, 21, 31 三套参数的组装效果

### 关键结果
- k=31 时性能最佳：N50 达 10,357 bp（提升 543%），覆盖度达 86.05%（+18%）
> 完整实验报告见：[`Reports/基因组调查与组装分析实验报告.pdf`](../reports/1_测序文件质量分析/基因组调查与组装分析实验报告.pdf)

### 为什么选择多k-mer测试？
拟南芥为高度纯合物种（杂合度<0.02%），但重复序列比例随k-mer变化显著（k=19时2.52%，k=31时1.62%）。大k-mer可更好跨越重复区，因此k=31组装的连续性和覆盖度最优。
