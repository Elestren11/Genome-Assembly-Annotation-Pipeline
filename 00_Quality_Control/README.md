## 测序数据质量控制

### Illumina PE150 数据
- 前8bp和后8bp存在固定引物/barcode残留
- 130bp后质量衰减明显（25%分位数降至Q20以下）
- Poly-A/C信号明显，提示建库偏好
- **处理策略**：修剪前端引物 + 切除尾部低质量 + 过滤poly-A/C尾

### PacBio HiFi 数据
- 单碱基质量极高（全读长>Q40）
- 但数据复杂度极低（仅500条reads），不适合定量分析
- **判断**：仅可用作组装可行性测试
