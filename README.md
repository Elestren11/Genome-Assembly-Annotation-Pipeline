# 从原始数据到生物学洞察：基因组全流程分析实战

本项目展示了从测序原始数据出发，完成**基因组组装、注释、及变异检测**的完整管道搭建与策略优化。所有结果基于课程项目，但独立完成了参数对比、工具选型和关键指标的量化评估。

## ⚠️ 关于分析的完整性

本仓库不仅展示成功结果，也保留了：
- 拟南芥注释中 **BUSCO 仅 15.2%** 的失败案例分析与改进方案
- 根瘤菌固氮基因未检出 SNP 的**阴性结果**及科学解释

> 完整的分析能力 = 发现问题 + 诊断原因 + 提出对策

## 🛠 技术栈
- **语言**：Shell (awk/sed)， Python（数据处理辅助）， R（统计分析与可视化，ggplot2，DESeq2）
- **二代组装**：FastQC, Trimmomatic, Jellyfish, GenomeScope, SOAPdenovo2, QUAST
- **三代组装**：wtdbg2, hifiasm
- **注释**：RepeatMasker, Augustus, BUSCO, Prokka
- **变异检测**：Bowtie2, SAMtools, BCFtools, snpEff
- **环境**：Conda, Linux 集群调度

## 📁 仓库结构
- `00_Quality_Control/`：Shell 文本处理 + FastQC 双平台数据质控对比
- `01_Genome_Assembly/`：基于 k-mer 预评估的二代组装策略优化
- `02_Assembly_Comparison/`：三代组装工具系统对比（含参数敏感性分析）
- `03_Annotation/`：真核与原核基因组注释（含拟南芥失败诊断）
- `04_Variant_Calling/`：SNP 检测、功能注释与热点基因分析
- `Reports/`：完整实验报告与关键结果截图（PDF），展示每步分析的细节与讨论
