## 根瘤菌SNP检测与功能注释

### 数据概况
- 参考基因组：5,035,299 bp，4,821个CDS
- 测序数据：7,864条双端reads
- 比对率：40.84%（样本间基因组差异可能是主因）

### SNP检测结果
| 质量等级 | 数量 | 占比 |
|:---|:---|:---|
| 高质量（Q≥100） | 687 | 81.01% |
| 中等质量（Q20-99） | 118 | 13.92% |
| 低质量（Q<20） | 43 | 5.07% |

### 功能注释
| 影响等级 | 数量 | 占比 |
|:---|:---|:---|
| HIGH（终止密码子获得） | 2 | 0.24% |
| MODERATE（错义突变） | 169 | 19.93% |
| LOW（同义突变） | 606 | 71.46% |
| MODIFIER（非编码区） | 71 | 8.37% |

### 固氮基因搜索——阴性结果
搜索nif/nod/fix/nitro基因家族：参考注释中nod基因4个、nitro基因10个，但VCF中均未发现SNP。固氮基因可能在进化中高度保守，或存在菌株特异性差异。阴性结果同样是科学发现。

### 局限性与拓展
- 比对率偏低（40.84%）是主要限制因素，需优化比对参数或检查参考基因组完整性
- 单一样本无法做群体遗传学分析
- 后续可设计GWAS方案：100-200菌株群体 + 标准化固氮表型测定
